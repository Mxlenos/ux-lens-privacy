🔍 UX-Lens Gizlilik Politikası

Son Güncelleme: Aralık 2024

Geliştirici: BusinessUp

İletişim: info@businessup.com.tr

📋 Genel Bakış

UX-Lens, web sitelerinin kullanıcı deneyimini analiz eden bir Chrome eklentisidir. Bu gizlilik politikası, eklentimizin hangi verileri topladığını ve nasıl kullandığını açıklar.

📊 Toplanan Veriler

🖼️ Ekran Görüntüleri

  • Ne: Aktif tarayıcı sekmesinin ekran görüntüsü
  • Neden: UX/UI analizi yapmak için
  • Nasıl İşlenir: Google Gemini API'sine gönderilir, analiz sonrası silinir
  • Saklanma: Geçici işlem, kalıcı depolama yok

🌐 HTML İçeriği

  • Ne: Web sayfasının HTML kodu ve DOM yapısı
  • Neden: Erişilebilirlik ve yapısal analiz için
  • Nasıl İşlenir: Google Gemini API'sine gönderilir
  • Saklanma: Geçici işlem, kalıcı depolama yok

🔑 API Anahtarı

  • Ne: Kullanıcının Google Gemini API anahtarı
  • Saklanma: Sadece kullanıcının tarayıcısında (chrome.storage.local)
  • Erişim: Sadece kullanıcı erişebilir
  • Paylaşım: Hiçbir şekilde paylaşılmaz veya gönderilmez

📄 Analiz Sonuçları

  • Ne: AI tarafından üretilen UX/UI analiz raporları
  • Saklanma: Kullanıcının tarayıcısında 30 dakika (önbellek)
  • Amaç: Hızlı erişim ve performans iyileştirmesi
  • Paylaşım: Hiçbir şekilde paylaşılmaz

🛡️ Veri Güvenliği

  • Tüm API çağrıları HTTPS üzerinden yapılır
  • API anahtarları sadece kullanıcının cihazında saklanır
  • Bizim sunucularımızda hiçbir kullanıcı verisi saklanmaz
  • Veriler sadece analiz için geçici olarak işlenir

🌐 Üçüncü Taraf Hizmetler

Google Gemini API: AI analizi için kullanılır. Google'ın kendi gizlilik politikası geçerlidir:

https://policies.google.com/privacy

👤 Kullanıcı Kontrolleri

  • API anahtarınızı istediğiniz zaman silebilirsiniz
  • Önbelleği temizleyebilirsiniz
  • Eklentiyi istediğiniz zaman kaldırabilirsiniz
  • Hiçbir zorunlu veri toplama yoktur
